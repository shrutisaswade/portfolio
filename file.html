Underwater Debries Detection using Machine Learning

The Underwater Debris Detection project leverages machine learning, specifically YOLO models, to identify debris in underwater environments. A dataset was created by capturing video data, converting it into images, and using these images for training the models. In addition, a hardware model was designed to support real-time debris detection. A website platform was developed to showcase the findings and facilitate data analysis. Various YOLO models (YOLO NAS, YOLOv5s, YOLOv6s, YOLOv8) were compared to optimize detection accuracy.

Tech Stack: YOLO NAS, YOLOv8, Machine Learning, Web Development, Hardware Design
